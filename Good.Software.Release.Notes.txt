About This Release
Version 3.2.0 of Good Software – Cloud Suite delivers significant improvements in platform 
performance, user interface, and system security. This release introduces a brand-new workflow 
builder, smarter dashboards, and enhanced logging and compliance capabilities.
It also addresses a series of user-reported bugs and lays the groundwork for transitioning from 
legacy features to newer, more scalable tools. This release is fully backward-compatible with version 
3.1.x.
Release Highlights
This section summarizes the most impactful additions and changes in this release.
 New Custom Workflow Builder – Visual editor to automate multi-step processes
 Advanced SAML 2.0 SSO Support – Greater security and compliance for enterprise clients
 Performance Optimizations – Faster API responses, reduced UI load time
 Critical Bug Fixes – Stability improvements for all supported platforms
 Feature Sunset Notices – Deprecation of legacy export and admin tools

Enhancements
This release improves existing functionality across the platform, particularly in terms of performance, 
UI consistency, and security.
Component Improvement
REST API v2 Reduced average response time by 40%. Pagination added to bulk endpoints.
UI/UX Consistent font usage, tighter layout spacing, and accessibility improvements (WCAG 
2.1 compliance).
Security OAuth 2.0 token expiration is now configurable. SAML sessions auto-terminate after 
inactivity.
Performance improvements will be most noticeable in data-heavy modules like reporting, API 
integrations, and logs.

Upgrade Instructions
Follow the steps below to upgrade your instance to version 3.2.0:
1. Backup your environment – Ensure both the database and filesystem are backed up.
2. Review deprecated features – Transition from deprecated tools like the CSV exporter.
3. Run the Installer – Use the Upgrade Package from the admin panel.
4. Post-upgrade checks – Run system diagnostics from the Admin Console > Health Check.
Downtime is expected to be under 10 minutes for standard installations.
